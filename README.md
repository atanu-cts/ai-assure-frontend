# ai-assure-frontend

## Install Packages

```
npm install
```

## Run the app (in dev env)

```
npm run dev
```

## Run Format fixed and checker

```
npm run format && npm run format:check
```

## Run Lint and Targeted Test

```
npm run lint:js && npm run lint:scss && npm run test -- src/server/home/controller.test.js
```
